Experiment 1b: Hyperlinks in HTML

Below are examples of hyperlinks demonstrating the <a> tag with href and target attributes:

  • Visit Google (opens in new tab)
  • Visit Wikipedia (opens in same tab)
  • Internal Link to page2.html (same folder)

Explanation:

  • href: Specifies the URL or path the link goes to.
  • target="_blank": Opens the link in a new browser tab or window.
  • target="_self": Opens the link in the same tab/window (default behavior).
